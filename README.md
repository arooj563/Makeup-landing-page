# Makeup Landing Page

This is a **simple HTML, CSS, and JavaScript landing page** for a makeup brand.  
It showcases products, services, and basic contact information.

---

## Features

- Clean and **responsive design** for desktop and mobile.  
- Sections include **Home, About, Products, and Contact**.  
- Interactive buttons and links.  
- Simple animations and hover effects.

---

## Technologies Used

- HTML  
- CSS  
- JavaScript
